# Utility-Tool-kit-By-Python
Simple README: How it Works
Code Logic (Simple Breakdown)
1. Functions (def)
Each tool is wrapped in a def block. This keeps the math logic separate from the menu logic.
2. The Main Manager (main)
The program uses a main () function to act as the "control center." It handles the menu, gets your input, and calls the right tool for the job.
3. Safety Net (try-except)
The program is "crash-proof." If a user accidentally types letters when the program expects a number, the try-except block catches the mistake and shows a friendly error instead of closing.
4. Loop (while)
The while True loop keeps the program open so you can use multiple tools in one session without having to restart the script.
